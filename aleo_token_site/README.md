# Aleo Token Site

- Aleo testnet3 상에 배포한 `token_3lzgy5.aleo` 프로그램을 이용할 수 있는 프론트엔드입니다.
- [token_3lzgy5.aleo](https://explorer.hamp.app/program?id=token_3lzgy5.aleo)

## 기능 구현

- [x] Public Mint

- [ ] Private Transfer

- [ ] Private to Public Transfer
